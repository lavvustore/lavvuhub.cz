Claude API
API klíč najdete na console.anthropic.com.
Klíč se ukládá pouze lokálně ve vašem prohlížeči a nikam se neodesílá.

Pokud test selže s CORS chybou: Otevřete terminál, přejděte do složky s tímto souborem a spusťte:
python3 -m http.server 8080
Pak otevřete http://localhost:8080/prekladac-newsletteru.html
